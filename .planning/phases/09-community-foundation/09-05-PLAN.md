**Pattern for Save to Shelf button:**

The button should:
- Only render in Tauri context (wrap in `{#if tauriMode}`)
- On mount: call `isInAnyCollection('artist', mbid)` to check if already saved — if any collections contain it, show a checkmark on the button
- On click: show a small dropdown (popover) with the list of collections from `collectionsState.collections`
- Each collection in dropdown: click to add item (calls `addToCollection(...)`) and close dropdown
- Bottom of dropdown: "New Shelf" inline — text input, on Enter creates the collection via `createCollection(name)` and immediately adds the item to it
- If item already in a collection, that collection row shows a checkmark (✓)
- No vanity metrics anywhere near the button

Load collections on demand: in the Tauri `onMount` block, call `if (!collectionsState.isLoaded) await loadCollections()`.

Add the Save to Shelf button in the artist header area (next to or below the FavoriteButton — place it after the existing artist name row, wrapped in `{#if tauriMode}`):

```svelte
{#if tauriMode}
  <div class="save-shelf-wrapper" style="position:relative;">
    <button
      class="save-shelf-btn"
      class:saved={savedInCollections.length > 0}
      onclick={() => showSaveDropdown = !showSaveDropdown}
      aria-label="Save to shelf"
    >
      {savedInCollections.length > 0 ? '✓ Saved' : '+ Save to Shelf'}
    </button>
    {#if showSaveDropdown}
      <div class="shelf-dropdown">
        {#each collectionsState.collections as col (col.id)}
          <button
            class="shelf-option"
            class:in-collection={savedInCollections.includes(col.id)}
            onclick={async () => {
              const { addToCollection } = await import('$lib/taste/collections.svelte');
              await addToCollection(col.id, 'artist', data.artist.mbid, data.artist.name, data.artist.slug);
              savedInCollections = [...savedInCollections, col.id];
              showSaveDropdown = false;
            }}
          >
            {col.name} {savedInCollections.includes(col.id) ? '✓' : ''}
          </button>
        {/each}
        <div class="new-shelf-inline">
          <input
            class="new-shelf-input-sm"
            type="text"
            bind:value={newShelfNameArtist}
            placeholder="New shelf..."
            onkeydown={async (e) => {
              if (e.key === 'Enter' && newShelfNameArtist.trim()) {
                const { createCollection, addToCollection } = await import('$lib/taste/collections.svelte');
                const id = await createCollection(newShelfNameArtist.trim());
                if (id) {
                  await addToCollection(id, 'artist', data.artist.mbid, data.artist.name, data.artist.slug);
                  savedInCollections = [...savedInCollections, id];
                }
                newShelfNameArtist = '';
                showSaveDropdown = false;
              }
            }}
          />
        </div>
      </div>
    {/if}
  </div>
{/if}
```
