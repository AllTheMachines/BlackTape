**Three node types — visual differentiation (Claude's discretion on exact design):**
- `genre` nodes: filled circle, CSS var `--color-primary` or similar
- `scene` nodes: diamond shape (SVG polygon with 4 points rotated 45°), distinct color
- `city` nodes: small circle with dashed border stroke

**Node interaction:**
- Click: call `onNodeClick(node)` if provided, else navigate directly: `import { goto } from '$app/navigation'; goto('/kb/genre/' + node.slug)`
- Hover: highlight node and its direct neighbors (add CSS class), dim others — same approach as StyleMap hover

**focusSlug:** When set, visually emphasize the node with that slug (larger radius, full opacity). Used when graph is in side-panel mode.

**Reactive to prop changes:** When `nodes` or `edges` props change (new subgraph loaded after click), re-run simulation and reassign layout state.

**SVG structure:** Same as StyleMap — viewBox="0 0 800 600", preserveAspectRatio="xMidYMid meet", responsive width 100%.

**DO NOT** wire `on('tick')` to reactive state — this causes continuous re-renders. The established pattern is: tick headlessly → stop → compute edge positions from settled nodes → assign to `$state` once.

```typescript
import type { PageServerLoad } from './$types';
import { D1Provider } from '$lib/db/d1-provider';
import { getStarterGenreGraph } from '$lib/db/queries';

export const load: PageServerLoad = async ({ platform }) => {
  const db = new D1Provider(platform!.env.DB);
  // No taste tags on web (no local profile) — use top-connected genres
  const graph = await getStarterGenreGraph(db, []);
  return { graph };
};
```

**`src/routes/kb/+page.ts`** (universal — same pattern as style-map/+page.ts):
```typescript
import type { PageLoad } from './$types';

export const load: PageLoad = async ({ data }) => {
  // Web: pass server data through unchanged
  if (typeof window === 'undefined' || !(window as any).__TAURI_INTERNALS__) {
    return data;
  }
  // Tauri: query local DB with taste tags for personalized starting graph
  try {
    const { getProvider } = await import('$lib/db/provider');
    const { getStarterGenreGraph } = await import('$lib/db/queries');
    const { tasteProfile } = await import('$lib/taste/profile.svelte');
    const db = await getProvider();
    const tasteTags = tasteProfile.tags.map(t => t.tag).slice(0, 5);
    const graph = await getStarterGenreGraph(db, tasteTags);
    return { graph };
  } catch {
    return data;
  }
};
```

Note: `tasteProfile` is the reactive $state object exported from `$lib/taste/profile.svelte`. It has a `.tags` array of `{ tag, weight, source }`. Extract tag strings via `.map(t => t.tag)`. No async call needed — the state is already loaded by the app shell. If tasteProfile.tags is empty (not yet loaded), we get an empty array and fall back to the top-connected genres naturally.
