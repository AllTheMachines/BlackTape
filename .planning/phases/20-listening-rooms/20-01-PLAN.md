  <action>
Create `src/lib/comms/listening-room.svelte.ts` mirroring the architecture of `sessions.svelte.ts`. This module handles ALL Nostr communication for listening rooms and holds reactive state.

**Architectural constraint (from sessions.svelte.ts):** This module MUST NEVER call any Tauri invoke(). All state is Svelte $state — garbage collected when leaveRoom() is called.

**Event kinds (from RESEARCH.md):**
- `kind:30311` (addressable, `#d` tag guaranteed) — room lifecycle open/close. Use `d: 'mercury-room-'+channelSlug` as unique identifier. Query: `{kinds:[30311], '#d':['mercury-room-'+sceneSlug]}`. This is the recommended choice over kind:10311 per RESEARCH.md Pitfall 3 — relay tag filter reliability is guaranteed for addressable events.
- `kind:20010` (ephemeral) — host sets active YouTube URL; all participants load it
- `kind:20011` (ephemeral) — jukebox suggestion (action: 'suggest' | 'retract' | 'reject')
- `kind:20012` (ephemeral) — presence heartbeat

**NDKKind double-cast** (required — enum doesn't include custom kinds):
```typescript
kinds: [20010, 20011, 20012] as unknown as NDKKind[]
```

**Interfaces to define:**
```typescript
export interface RoomParticipant {
  pubkey: string;
  displayName: string;   // Nostr kind:0 display_name, fallback: pubkey.slice(0,8)+'...'
  avatarSeed: string;    // pubkey (DiceBear seed)
  lastSeen: number;      // unix timestamp
}

export interface QueueItem {
  id: string;            // suggestion event ID
  senderPubkey: string;
  youtubeUrl: string;    // original URL (not embed URL)
  submittedAt: number;   // event.created_at for FIFO order
  state: 'pending' | 'retracted' | 'approved' | 'rejected';
}
```

**roomState ($state object):**
```typescript
export const roomState = $state({
  isInRoom: false,
  isHost: false,
  channelSlug: null as string | null,
  hostPubkey: null as string | null,
  activeVideoUrl: null as string | null,  // embed URL (youtube-nocookie.com/embed/...)
  queue: [] as QueueItem[],
  participants: {} as Record<string, RoomParticipant>,  // use Record not Map (Svelte $state reactivity)
  myPendingSuggestionId: null as string | null,
  loading: false,
  error: null as string | null,
});
```

Note: Use `Record<string, RoomParticipant>` instead of `Map` — Svelte 5 `$state` tracks plain objects deeply but Map mutation does not trigger reactivity.

**Functions to implement:**

1. `openRoom(channelSlug: string): Promise<void>` — Publishes kind:30311 with `{status:'open', hostPubkey: ndkState.pubkey, channelSlug}`, sets roomState.isHost=true, calls _subscribeToRoomEvents(), startHeartbeat(), startPresenceCleanup(). Check if user is already in a room first.

2. `joinRoom(channelSlug: string, hostPubkey: string, currentVideoUrl?: string): Promise<void>` — Sets roomState from known room state, calls _subscribeToRoomEvents(), startHeartbeat(), startPresenceCleanup(). If currentVideoUrl provided, sets roomState.activeVideoUrl.

3. `leaveRoom(): Promise<void>` — If host: publishes kind:30311 with `{status:'closed'}`. Stops all timers and subscriptions. Resets roomState to defaults.

4. `setActiveVideo(youtubeUrl: string): Promise<void>` — Host only. Validates with `youtubeEmbedUrl()` from `$lib/embeds/youtube.js`. Publishes kind:20010 with `{youtubeUrl, embedUrl}`. Optimistic update: `roomState.activeVideoUrl = embedUrl`.

5. `submitSuggestion(youtubeUrl: string): Promise<void>` — Guest only. Checks `roomState.myPendingSuggestionId === null`. Validates URL. Publishes kind:20011 with `{youtubeUrl, action:'suggest'}`. Sets `roomState.myPendingSuggestionId = event.id`.

6. `retractSuggestion(): Promise<void>` — Guest only. Publishes kind:20011 with `{action:'retract', eventId: roomState.myPendingSuggestionId}` + `['e', suggestionEventId]` tag. Clears `roomState.myPendingSuggestionId`.

7. `approveQueueItem(item: QueueItem): Promise<void>` — Host only. Calls `setActiveVideo(item.youtubeUrl)` (which publishes kind:20010). Publishes kind:20011 with `{action:'reject', eventId:item.id}` is NOT needed — approval is implicit (the kind:20010 clears queue locally on all clients when they see the new active video match a suggestion). Actually: after publishing kind:20010, mark item.state='approved' locally and clear queue items with state approved/rejected on render.

8. `rejectQueueItem(item: QueueItem): Promise<void>` — Host only. Publishes kind:20011 with `{action:'reject', eventId:item.id}` + `['e', item.id]` tag. Subscribers mark that item state='rejected'.

**Internal subscription `_subscribeToRoomEvents(channelSlug: string)`:**
- Subscribe with `closeOnEose: false` to kinds [20010, 20011, 20012] filtered by `'#t': ['mercury', channelSlug]`
- kind:20010 handler: parse content, validate embedUrl, set `roomState.activeVideoUrl = embedUrl`. Also check queue — if any queue item's youtubeUrl matches the new active URL, mark it 'approved'.
- kind:20011 handler: parse action. 'suggest': add QueueItem to queue (use created_at for submittedAt, sort by submittedAt ascending). 'retract'/'reject': find item by `['e', id]` tag, mark state='retracted'/'rejected'. Deduplicate by event.id. Update `myPendingSuggestionId` if retract event matches.
- kind:20012 handler: upsert participant in `roomState.participants[event.pubkey]`. Set lastSeen = Math.floor(Date.now()/1000). Attempt best-effort profile fetch: `const user = ndk.getUser({pubkey: event.pubkey}); user.fetchProfile().then(p => { if (p?.displayName || p?.name) roomState.participants[event.pubkey].displayName = p.displayName ?? p.name ?? ...; }).catch(()=>{});`
- Store sub in module-level `_roomSub` variable (single subscription, not a Map like sessions.svelte.ts since there's only one room at a time)

**Heartbeat (30s interval):**
```typescript
let _heartbeatTimer: ReturnType<typeof setInterval> | null = null;
let _cleanupTimer: ReturnType<typeof setInterval> | null = null;

function startHeartbeat(channelSlug: string): void {
  _sendHeartbeat(channelSlug);
  _heartbeatTimer = setInterval(() => _sendHeartbeat(channelSlug), 30_000);
}

async function _sendHeartbeat(channelSlug: string): Promise<void> {
  const { ndk } = ndkState;
  if (!ndk) return;
  const { NDKEvent } = await import('@nostr-dev-kit/ndk');
  const event = new NDKEvent(ndk);
  event.kind = 20012 as unknown as NDKKind;
  event.content = '';
  event.tags = [['t','mercury'], ['t', channelSlug], ['expiration', String(Math.floor(Date.now()/1000)+90)]];
  await event.publish();
}

function startPresenceCleanup(): void {
  _cleanupTimer = setInterval(() => {
    const cutoff = Math.floor(Date.now()/1000) - 75;
    for (const pubkey of Object.keys(roomState.participants)) {
      if (roomState.participants[pubkey].lastSeen < cutoff) {
        delete roomState.participants[pubkey];
      }
    }
  }, 10_000);
}
```

**Cleanup in leaveRoom():**
```typescript
if (_heartbeatTimer) { clearInterval(_heartbeatTimer); _heartbeatTimer = null; }
if (_cleanupTimer) { clearInterval(_cleanupTimer); _cleanupTimer = null; }
if (_roomSub) { _roomSub.stop(); _roomSub = null; }
```
