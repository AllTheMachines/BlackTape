4. In `onMount`, after the existing `loadSceneFollows()` call, add room check (Tauri-only, best-effort):
```typescript
// Check if a listening room is active for this scene
if (data.scene) {
  try {
    const result = await checkActiveRoom(data.scene.slug);
    if (result.active) {
      roomStatus = 'active';
      roomHostPubkey = result.hostPubkey ?? null;
    } else {
      roomStatus = 'none';
    }
  } catch {
    roomStatus = 'none'; // best-effort — fail silently
  }
}
```

5. Add handler for starting a room:
```typescript
async function handleStartRoom() {
  if (!data.scene) return;

}
```
