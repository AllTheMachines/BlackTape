    Create `src/lib/components/ArtistSummary.svelte` as a new file. This component must implement the full state machine for the AI summary section.

    **Props interface:**
    - `artistMbid: string` ŌĆö used as cache key for taste.db
    - `artistName: string` ŌĆö passed to the prompt
    - `artistTags: string` ŌĆö comma-separated tags string (data.artist.tags from page)
    - `releases: Array<{ title: string; year: number | null; type: string }>` ŌĆö from data.releases

    **State variables (Svelte 5 $state):**
    - `summaryText: string | null` ŌĆö null = no cache, hidden section
    - `generatedAt: number | null` ŌĆö Unix seconds from DB row
    - `isGenerating: boolean` ŌĆö true while AI call is in flight

    **Constants:**
    - `SUMMARY_TTL_SECONDS = 30 * 24 * 60 * 60` (30 days)

    **Helper functions:**
    - `isSummaryStale(ts: number): boolean` ŌĆö returns true if (now - ts) > TTL
    - `formatRelativeTime(ts: number): string` ŌĆö returns "Generated today" / "Generated yesterday" / "Generated N days ago"

    **generateSummary() async function:**
    1. Import `getAiProvider` from `$lib/ai/engine` ŌĆö if null, return early (silently)
    2. Set `isGenerating = true`
    3. In try block:
       a. Import `artistSummaryFromReleases` from `$lib/ai/prompts`
       b. Build prompt with artistName, releases, artistTags
       c. Call `provider.complete(prompt.user, { systemPrompt: prompt.system, temperature: 0.3, maxTokens: 200 })`
       d. If result is truthy and non-empty after trim:
          - Import invoke from `@tauri-apps/api/core`
          - Call `invoke('save_artist_summary', { artistMbid, summary: result.trim() })`
          - Update `summaryText` and `generatedAt` in state
    4. catch block: do nothing (silent fail per spec)
    5. finally: `isGenerating = false`

    **onMount IIFE pattern (same as ArtistStats.svelte):**
    ```
    onMount(async () => {
      (async () => {
        try {
          const { invoke } = await import('@tauri-apps/api/core');
          const cached = await invoke<{ summary: string; generated_at: number } | null>(
            'get_artist_summary', { artistMbid }
          );
          if (cached) {
            summaryText = cached.summary;
            generatedAt = cached.generated_at;
            if (isSummaryStale(cached.generated_at) && aiState.autoGenerateOnVisit) {
              generateSummary(); // intentionally not awaited ŌĆö background refresh
            }
          } else if (aiState.autoGenerateOnVisit && getAiProvider()) {
            generateSummary(); // no cache, auto-generate if opted in
          }
          // else: no cache, auto-generate off ŌĆö section stays hidden
        } catch {
          // Silent
        }
      })();
    });
    ```
