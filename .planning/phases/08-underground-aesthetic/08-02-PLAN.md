Create `src/lib/components/PanelLayout.svelte`:

This component wraps page content in PaneForge panels based on the active template. It receives template ID and slot content via props/snippets.

```svelte
<script lang="ts">
    import { PaneGroup, Pane, PaneResizer } from 'paneforge';
    import { getTemplate } from '$lib/theme/templates';
    import type { LayoutTemplate } from '$lib/theme/templates';
    import type { Snippet } from 'svelte';

    let {
        template = 'default' as LayoutTemplate,
        children,
        sidebar,
        context
    }: {
        template?: LayoutTemplate;
        children: Snippet;
        sidebar?: Snippet;
        context?: Snippet;
    } = $props();

    let config = $derived(getTemplate(template));
    let hasPanels = $derived(template !== 'default' && template !== 'minimal');
</script>
```

**Rendering logic:**

- If `template === 'default'` or `template === 'minimal'`: render `{@render children()}` directly (no PaneForge). Minimal template gets a CSS class `minimal-layout` for future overlay control styling.

- If `template === 'cockpit'`: render a horizontal PaneGroup with autoSaveId `"mercury-cockpit"`:
  - Left Pane (sidebar slot): defaultSize=22, minSize=15. Renders `{@render sidebar?.()}` if sidebar snippet provided, else empty placeholder div with class `panel-empty`.
  - PaneResizer with class `panel-resizer`
  - Center Pane (main): defaultSize=56, minSize=30. Renders `{@render children()}`.
  - PaneResizer with class `panel-resizer`
  - Right Pane (context slot): defaultSize=22, minSize=15. Renders `{@render context?.()}` if context snippet provided, else empty placeholder div.

- If `template === 'focus'`: render a horizontal PaneGroup with autoSaveId `"mercury-focus"`:
  - Center Pane (main): defaultSize=70, minSize=40. Renders `{@render children()}`.
  - PaneResizer with class `panel-resizer`
  - Right Pane (context): defaultSize=30, minSize=15. Renders `{@render context?.()}` or empty placeholder.

**Styling:**

- `.panel-resizer`: width 4px, background transparent, transition background 0.15s. On hover: background `var(--border-hover)`. Cursor: col-resize.
- `.panel-empty`: full height, display flex, align-items center, justify-content center, color `var(--text-muted)`, font-size 0.75rem. Shows "Drag to resize" text as placeholder.
- Each pane content area: `overflow-y: auto`, `height: 100%`, `padding: var(--space-sm)`.
- The PaneGroup wrapper: `height: calc(100vh - var(--header-height))` for full viewport below header. When player is active, subtract player height too (pass `hasPlayer` prop or use CSS class).

Add a `hasPlayer` boolean prop (default false) that adjusts the height calculation:
- Without player: `height: calc(100vh - var(--header-height))`
- With player: `height: calc(100vh - var(--header-height) - var(--player-height))`

IMPORTANT: Do NOT import from `@tauri-apps/api` in this component. PanelLayout is a pure UI component. Platform gating (isTauri check) happens in +layout.svelte (Plan 03).
