Create `src/lib/theme/engine.svelte.ts` (uses .svelte.ts extension for $state runes — project convention):

1. Import `tasteTagsToHue`, `generatePalette`, `PALETTE_PROPERTIES` from `./palette`.
2. Import `loadPreferences`, `savePreference`, `DEFAULT_PREFERENCES` from `./preferences`.
3. Import type `UserPreferences`, `ThemeMode` from `./preferences`.

4. Export reactive state:
```typescript
export const themeState = $state({
    mode: 'default' as ThemeMode,
    manualHue: 220,
    computedHue: 0,
    isLoaded: false
});
```

5. Export `applyPalette(palette: Record<string, string>): void` — sets each property on `document.documentElement.style`. Must set ALL properties from PALETTE_PROPERTIES. Wrap in `typeof document !== 'undefined'` guard for SSR safety.

6. Export `clearPalette(): void` — removes all PALETTE_PROPERTIES from `document.documentElement.style`. Same SSR guard.

7. Export `async function loadThemePreferences(): Promise<void>` — calls `loadPreferences()`, updates `themeState.mode`, `themeState.manualHue`, sets `themeState.isLoaded = true`.

8. Export `function computeAndApplyTheme(tasteTags: Array<{ tag: string; weight: number }>): void`:
   - If `themeState.mode === 'default'`: call `clearPalette()`, return.
   - If `themeState.mode === 'taste'`: compute hue via `tasteTagsToHue(tasteTags)`, store in `themeState.computedHue`, generate palette, call `applyPalette`.
   - If `themeState.mode === 'manual'`: use `themeState.manualHue`, generate palette, call `applyPalette`.

9. Export `async function setThemeMode(mode: ThemeMode): Promise<void>` — updates `themeState.mode`, calls `savePreference('theme_mode', mode)`.

10. Export `async function setManualHue(hue: number): Promise<void>` — clamps 0-360, updates `themeState.manualHue`, calls `savePreference('theme_manual_hue', String(hue))`.
