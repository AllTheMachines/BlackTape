**Create `src/lib/components/ControlBar.svelte`:**

A dense horizontal toolbar that lives inside panel sidebars (or at the top of main content in minimal/default templates). Contains interactive filter controls.

Props:
- `variant: 'sidebar' | 'inline'` (default 'inline') — sidebar variant is vertical-stacked for panel sidebars; inline is horizontal row.

Content:
- **Quick tag filter**: A compact text input with placeholder "Filter tags..." that filters the current page's tag list. Uses `$state` for the filter value. Styled with small font (0.75rem), no border on focus except bottom accent line.
- **View toggle**: Two icon buttons (grid/list) that toggle between grid and list view. Emits a `viewChange` event via callback prop `onViewChange?: (view: 'grid' | 'list') => void`. Styled as icon-only toggle buttons with active state highlighting.
- **Sort dropdown**: A native `<select>` with options: "Relevance", "Name A-Z", "Name Z-A", "Most Niche", "Newest". Font size 0.75rem, background `var(--bg-elevated)`, color `var(--text-secondary)`, border `var(--border-default)`.

Styling:
- `.control-bar`: display flex, gap var(--space-xs), align-items center, padding var(--space-xs) var(--space-sm), background var(--bg-surface), border-bottom 1px solid var(--border-subtle), font-size 0.75rem.
- `.control-bar.sidebar`: flex-direction column, align-items stretch, padding var(--space-sm).
- All controls: compact, dense, monospace hints (sort dropdown uses `var(--font-mono)` for values).
- Toggle buttons: 24x24px, border-radius 3px, transition background 0.1s. Active: background var(--bg-hover), color var(--text-accent).
- Input and select heights: 24px.

NOTE: ControlBar is a UI component only — it does not manage global state. The page that includes it wires callbacks. This keeps it reusable across different pages and panel slots.
