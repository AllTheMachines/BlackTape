  - Strip trailing parenthetical qualifiers like "(Remastered)" or "[Deluxe]"
  - Trim again
- Implement `matchArtistToIndex(artistName: string): Promise<ArtistResult | null>`:
  - Get normalized name via `normalizeArtistName()`
  - Use `getProvider()` from `$lib/db/provider` to get the DbProvider
  - Call `searchArtists(db, normalizedName, 5)` from `$lib/db/queries`
  - From results, find best match:
    - Exact match (case insensitive) on name → return immediately
    - Otherwise return first result if rank is decent (trusting FTS5)
  - Return null if no results
  - Wrap in try/catch — matching is best-effort, never blocks playback
- Implement `getRelatedArtists(matchedArtist: ArtistResult): Promise<ArtistResult[]>`:
  - If the matched artist has tags, search by the first (most prominent) tag
  - Call `searchByTag(db, firstTag, 10)` from queries.ts
  - Filter out the matched artist itself
  - Return top 5 results
  - This gives a "related artists" section powered by shared tags
