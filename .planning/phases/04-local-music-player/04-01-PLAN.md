**src-tauri/src/scanner/mod.rs** — Create scanner module with Tauri commands:
- `pub mod metadata;`
- Define `ScanProgress` struct: `scanned: u32`, `total: u32`, `current_file: String`. Derive `Clone, Serialize`.
- Define `LibraryState` struct that wraps `Mutex<rusqlite::Connection>`. This will be Tauri managed state.
- Implement `#[tauri::command] async fn scan_folder(path: String, on_progress: Channel<ScanProgress>, state: tauri::State<'_, LibraryState>) -> Result<u32, String>`:
  - Use `walkdir::WalkDir::new(&path)` to collect all files
  - Filter using `metadata::is_supported_audio()`
  - Collect into Vec first to know total count
  - Iterate, calling `metadata::read_track_metadata()` for each
  - Get file modification time via `std::fs::metadata(path).modified()` → convert to unix timestamp
  - Insert into library.db via `library::db::insert_track()`
  - Send progress via `on_progress.send()` every 50 files (not every file — avoids IPC flood)
  - Return total count of successfully scanned tracks
- Implement `#[tauri::command] fn get_library_tracks(state: tauri::State<'_, LibraryState>) -> Result<Vec<LocalTrack>, String>`:
  - Delegate to `library::db::get_all_tracks()`
- Implement `#[tauri::command] fn get_music_folders(state: tauri::State<'_, LibraryState>) -> Result<Vec<MusicFolder>, String>`:
  - Delegate to `library::db::get_music_folders()`
- Implement `#[tauri::command] fn add_music_folder(path: String, state: tauri::State<'_, LibraryState>) -> Result<(), String>`:
  - Delegate to `library::db::add_music_folder()`
- Implement `#[tauri::command] fn remove_music_folder(path: String, state: tauri::State<'_, LibraryState>) -> Result<(), String>`:
  - Delete from music_folders table
  - Delete all local_tracks where path starts with the folder path

    ```rust
    .setup(|app| {
        let app_data = app.path().app_data_dir().expect("failed to get app data dir");
        std::fs::create_dir_all(&app_data).ok();
        let conn = library::db::init_library_db(&app_data)
            .expect("failed to init library db");
        app.manage(scanner::LibraryState(std::sync::Mutex::new(conn)));
        Ok(())
    })
    ```
