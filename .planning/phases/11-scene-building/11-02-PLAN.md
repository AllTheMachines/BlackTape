**src/lib/scenes/types.ts** — Define all interfaces for the scenes module:

```typescript
export interface DetectedScene {
    slug: string;           // dominant tag as URL slug
    name: string;           // dominant tag, human-readable (capitalize, replace hyphens with spaces)
    tags: string[];         // all tags in the cluster, niche-first
    artistMbids: string[];  // MBIDs of artists in the scene (up to 20)
    listenerCount: number;  // count of user's own favorites in this scene
    isEmerging: boolean;    // true = novel tag combo not in known KB genres
    detectedAt: number;     // unix timestamp
}

export interface SceneArtist {
    mbid: string;
    name: string;
    slug: string;
    country: string | null;
    tags: string | null;
    isSuggested?: boolean;  // true if community-suggested, not AI-detected
}

export interface SceneSuggestion {
    artistMbid: string;
    artistName: string;
    suggestedAt: number;
}

export interface PartitionedScenes {
    active: DetectedScene[];    // established scenes (listenerCount > 2, not emerging)
    emerging: DetectedScene[];  // novel scenes (isEmerging OR listenerCount <= 2)
}
```

**src/lib/scenes/detection.ts** — The detection algorithm. Use the dynamic import / getInvoke pattern for Tauri calls (same as src/lib/library/scanner.ts or src/lib/taste/signals.ts). Use getProvider() for mercury.db queries (same as src/lib/db/queries.ts usage in discover page).

Implement these exported functions:

1. **`findTagClusterSeeds(db: DbProvider)`** — Query tag_cooccurrence with niche filters:
```sql
SELECT tc.tag_a, tc.tag_b, tc.shared_artists
FROM tag_cooccurrence tc
JOIN tag_stats ts_a ON ts_a.tag = tc.tag_a
JOIN tag_stats ts_b ON ts_b.tag = tc.tag_b
WHERE ts_a.artist_count < 200
  AND ts_b.artist_count < 200
  AND tc.shared_artists >= 5
ORDER BY tc.shared_artists DESC
LIMIT 200
```
Returns `Array<{ tag_a: string; tag_b: string; shared_artists: number }>`.

2. **`groupTagPairsIntoClusters(seeds)`** — Group overlapping tag pairs into clusters. Two pairs belong in the same cluster if they share at least one tag. Use a union-find / simple iterative merge: for each seed, check if tag_a or tag_b is already in an existing cluster's tag set; if yes, merge into that cluster; if no, create a new cluster. A cluster is `{ tags: Set<string> }`. Return clusters as `Array<{ tags: string[] }>` sorted by size desc. Cap at 50 clusters.

3. **`getClusterArtists(db: DbProvider, tags: string[])`** — Get artists that have ALL tags in the cluster (up to 3 tags; if cluster has more, use top 3 by niche):
```sql
SELECT a.id, a.mbid, a.name, a.slug, a.country,
       at1.count as primary_count
FROM artists a
JOIN artist_tags at1 ON at1.artist_id = a.id AND at1.tag = ?
JOIN artist_tags at2 ON at2.artist_id = a.id AND at2.tag = ?
ORDER BY at1.count DESC
LIMIT 20
```
For 3-tag clusters add a third JOIN. For 1-tag clusters, use a single JOIN. Returns `SceneArtist[]`.

4. **`isNovelTagCombination(tags: string[], db: DbProvider)`** — Returns true if none of the top 3 tags appear in the genres table as mb_tag:
```sql
SELECT COUNT(*) as cnt FROM genres WHERE mb_tag IN (?, ?, ?)
```
Returns `Promise<boolean>` — true = novel/emerging.

5. **`validateListenerOverlap(artistMbids: string[])`** — Tauri-only. Gets user's favorite_artists via invoke, returns count of how many appear in the candidate scene. If not in Tauri context, returns 0. Minimum viable scene = 2 of the user's favorites appear in scene artists.

6. **`detectScenes()`** — Main entry point (Tauri only — guard with isTauri() at top; return [] immediately on web):
```
1. getProvider() for mercury.db
2. findTagClusterSeeds(db)
3. groupTagPairsIntoClusters(seeds) → clusters
4. For each cluster (first 30 to limit execution time):
   a. getClusterArtists(db, cluster.tags) → artists
   b. if artists.length < 3: skip (not a scene)
   c. validateListenerOverlap(artists.map(a => a.mbid)) → overlap
   d. isNovelTagCombination(cluster.tags, db) → emerging
   e. dominantTag = cluster.tags[0]
   f. Push DetectedScene to results
5. save_detected_scenes via invoke (cache to taste.db)
6. Return results
```

7. **`partitionScenes(scenes: DetectedScene[])`** — Returns PartitionedScenes:
- emerging: scenes where isEmerging === true OR listenerCount <= 2
- active: scenes where !isEmerging AND listenerCount > 2
- Both arrays shuffled with Fisher-Yates (same as crate dig random — copy that pattern)

8. **`loadCachedScenes()`** — Tauri only. invoke get_detected_scenes, parse JSON fields (tags and artist_mbids are stored as JSON strings in Rust), return DetectedScene[]. Returns [] if not in Tauri.

Use `import { isTauri } from '$lib/platform'` for platform guards.
Use `import type { DbProvider } from '$lib/db/provider'` for typing.
All Tauri invoke calls use the `getInvoke` dynamic import pattern — check src/lib/library/ or src/lib/ai/ for the exact import.

**src/lib/scenes/state.svelte.ts** — Reactive state module (.svelte.ts extension required for $state runes). Follow the pattern from src/lib/theme/palette.svelte.ts or similar .svelte.ts modules:

```typescript
import type { DetectedScene, PartitionedScenes } from './types';

interface ScenesState {
    scenes: DetectedScene[];
    partitioned: PartitionedScenes;
    isLoaded: boolean;
    isDetecting: boolean;
    error: string | null;
}

export const scenesState = $state<ScenesState>({
    scenes: [],
    partitioned: { active: [], emerging: [] },
    isLoaded: false,
    isDetecting: false,
    error: null
});

export async function loadScenes(forceDetect = false): Promise<void> {
    if (scenesState.isDetecting) return; // idempotent guard
    scenesState.isDetecting = true;
    scenesState.error = null;

    try {
        const { loadCachedScenes, detectScenes, partitionScenes } = await import('./detection');

        // Try cache first (unless forceDetect)
        let scenes: DetectedScene[] = [];
        if (!forceDetect) {
            scenes = await loadCachedScenes();
        }

        // If cache empty or forced, run detection
        if (scenes.length === 0 || forceDetect) {
            scenes = await detectScenes();
        }

        scenesState.scenes = scenes;
        scenesState.partitioned = partitionScenes(scenes);
        scenesState.isLoaded = true;
    } catch (e) {
        scenesState.error = e instanceof Error ? e.message : 'Scene detection failed';
        scenesState.isLoaded = true; // still "loaded" — just empty
    } finally {
        scenesState.isDetecting = false;
    }
}
```
