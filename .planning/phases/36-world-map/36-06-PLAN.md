**Part A: Artist page — add geo check to load function**

Read `src/routes/rabbit-hole/artist/[slug]/+page.ts` first. Add a geocoordinates check to the existing load function. The simplest approach is a direct SQL query rather than calling `getGeocodedArtists` (which loads up to 50K records just to check one artist):

```typescript
// In the existing load function, after artist is fetched, add:
let hasGeocoordinates = false;
if (artist) {
  const geoRow = await db.get<{ has_geo: number }>(
    `SELECT (city_lat IS NOT NULL AND city_lat != 0) as has_geo FROM artists WHERE id = ?`,
    artist.id
  );
  hasGeocoordinates = (geoRow?.has_geo ?? 0) === 1;
}

return { artist, similarArtists, links: linksRaw, hasGeocoordinates };
```
