## User Setup Required

**External services require manual configuration before affiliate commissions are earned:**

| Service | Env Var | Where to get it | Notes |
|---------|---------|-----------------|-------|
| Amazon Associates | `AFFILIATE_AMAZON_TAG` | [affiliate-program.amazon.com](https://affiliate-program.amazon.com) → Account → Tracking IDs | Free, immediate approval |
| Apple Performance Partners | `AFFILIATE_APPLE_TOKEN` | [performance-partners.apple.com](https://performance-partners.apple.com) → Partnerize → Publisher → Integration → at= token | Gated application, may take days to weeks |
| Apple Performance Partners | `AFFILIATE_APPLE_CAMPAIGN` | Choose any short label (e.g. `mercury`) | Already pre-set in `.dev.vars` |

**For local dev:** Fill values in `.dev.vars` — used automatically by `npx wrangler pages dev`.

**For production:** Add the same vars to Cloudflare Pages dashboard → Settings → Environment Variables.

Without these values, buy links remain functional search fallback links (with `?` indicator) — no commissions, but no broken experience for users.
