## Decisions Made
- **thresholdFired resets on two events:** `loadedmetadata` (new track src assigned) and `play` when `currentTime < 1` (repeat-one restart). This covers the two distinct ways a listen can restart without creating false double-counts from seeking backward past the threshold.
- **Playback source is lowest priority:** In the merge loop, `existing.source` wins over `'playback'`. Tags from library and favorites carry more signal weight than passive listening history. Manual tags survive recompute entirely (not cleared).
- **5-play activation gate:** `computeTasteFromPlayHistory()` returns early (empty map) if total play count is below 5. This prevents noise from accidental plays influencing taste before the user has genuinely engaged with the player.
