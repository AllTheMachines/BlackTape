Brand Overview

Identity
Mercury
Codename — Real name TBD · Change once in src/lib/config.ts
Core Mission
A music discovery engine that rewards the niche. Indexes open databases, embeds players from where music already lives. Uniqueness is discoverability.
What It's Not
Not a platform. Not a streaming service. No audio hosting. No tracking. No algorithmic manipulation. A discovery tool, not a retention tool.

Direct Sparse Nerdy No fluff Open source Anti-corporate

Cassette

Graphics

The cassette graphic sits in the player bar track-info area. It's a layered SVG system: a static cassette body SVG behind two live spinning reel SVGs.

Live Demo — click to toggle play
Layer structure
① .cassette-body — static SVG, position: absolute, behind
② .reel × 2 — live SVGs, normal flex flow, on top
③ .reel animates spin only when .cassette-reels.playing
Click the cassette to toggle spin
Reel — isolated
viewBox: 0 0 20 20, rendered 29×29px
Outer ring: r=8.5, fill-opacity=0.08
Hub: 32-pt polygon, 8 notches, r outer=4 / notch r=3
Spindle hole: r=1.2, fill=var(--bg-3)

Retro FX Ideas

Graphics

Micro-details that add to the BlackTape aesthetic. "Little animated stuff and funny graphics everywhere… not big, just here and there… that is like the dot on the eye."

Scanlines
Repeating horizontal lines at 4px intervals. CSS-only, ~18% opacity. Applied to panel headers or sidebar. Evokes CRT monitor / old TV screen.
CSS · Static · Sidebar / panel headers
Film Grain / Noise
Animated perlin-style noise overlay on bg panels. Very subtle (5–8% opacity). Adds tactile depth — like a worn tape surface. Canvas-driven, ~24fps.
Canvas · Animated · Background panels
VU Meter Bars
Animated amber bars that pulse while a track plays. 5–7 bars, CSS keyframes. Could sit next to the track title in the player bar or in the sidebar.
CSS · Animated · Player bar / sidebar
Tape Counter
0 4 8 3
Retro mechanical counter style for track time. Monospaced digits with glow. Could replace or accompany the current time display. Each digit "ticks" like an odometer.
CSS + JS · Animated · Player seek row
CRT Phosphor Glow
Now Playing
Amber glow (text-shadow + box-shadow) on accent elements — track title, active nav item, playing indicator. Subtle. Like a CRT screen in a dark room.
CSS · Static / hover · Accent elements
Tape Type Badge
TYPE II CHROME C-90
Retro cassette classification badges. Could tag audio quality, source platform, or release format. Tiny, uppercase, monospaced border treatment.
CSS · Static · Cards / track rows
Blinking LED Indicator
REC
PLAY
Small glowing LED dot. Amber = recording/active. Green = playing. Could sit in the player bar or sidebar header to show playback state.
CSS · Animated · Player bar / sidebar
Pixel Corner Brackets
Tiny L-shaped corner accents on cards or the now-playing panel. Like targeting reticles or old UI chrome. Accent color, very subtle opacity. Seen on this card right now.
CSS · Static · Cards / now-playing panel
Idle Waveform / Tape Hiss
A slow, flat animated waveform that pulses when nothing is playing — like tape hiss. When a track plays it becomes more active. Could replace the static cassette reels when idle.
SVG + CSS · Animated · Player bar idle state
