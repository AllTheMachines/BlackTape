# Mercury (working title)

A music search engine with taste.

Indexes all music from open databases. Embeds players from wherever music already lives. Lets you discover through atomic tags. No algorithms. No ads. No bullshit.

**Status:** Early development. The search engine doesn't work yet. We're building the foundation.

## What this will be

- 2.6 million artists indexed on day one (from MusicBrainz, CC0)
- Embedded players from Bandcamp, Spotify, SoundCloud, YouTube
- Atomic tags for real discovery (not "electronic" but "dark ambient / granular synthesis / Berlin school")
- Every artist discoverable by default — opt out, not opt in
- Open source so no company can shut it down
- $0 infrastructure (Cloudflare free tier)

## Development

```bash
npm install
npm run dev
```
